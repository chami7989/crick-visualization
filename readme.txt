# Cricket Data Analysis Tool

## Overview

This project is a cricket data analysis tool that utilizes Python, HTML, CSS, and JSON to extract, analyze, and present cricket statistics. It's designed to provide insights and visualizations based on cricket match data.

## Features

- **Top Scoring Batsmen**: Get a list of the highest-scoring batsmen based on the provided JSON data.
- **Data Analysis**: Utilize Python to process and analyze the cricket statistics.
- **HTML Presentation**: Present the analyzed data in an interactive HTML format with CSS for styling.

## Getting Started

### Prerequisites

Before you begin, ensure you have met the following requirements:

- Python installed (version X.X)
- Basic knowledge of HTML and CSS
- Understanding of JSON data format

### Installation

1. Clone this repository: `git clone https://github.com/yourusername/cricket-data-analysis.git`
2. Navigate to the project directory: `cd cricket-data-analysis`
3. Run the Python script to analyze the data: `python analyze_data.py`
4. Open `index.html` in your web browser to view the results.

## Usage

1. **Data Input**: Replace the sample JSON data with your cricket match data in the `data.json` file.
2. Run the Python script to perform the analysis.
3. Open `index.html` in your web browser to see the top-scoring batsmen and other statistics.
